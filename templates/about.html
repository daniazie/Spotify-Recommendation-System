About

In this project, I created a music recommendation system using the Spotify API based on a user's top tracks within the last 4 weeks. Users can (and quite frankly, most likely will) get song recommendations from all over the world.

How It Works

After getting a user's top tracks, I used this dataset from Kaggle to recommend songs by calculating a vector with songs from a user's top tracks that overlaps with those in the dataset, then using cosine similarity to calculate the similarity between songs in the vector and those in the dataset. 15 songs with the highest similarity are then recommended to the user.

Users can also listen to previews of the songs (except for those that do not have preview urls) and create a playlist with those 15 songs if they'd like.

However, the dataset I used only features songs from 1921-2020, so users will be unable to be recommended songs that were released after the year 2020. Also, since the calculation is done purely based on audio features, users will also get songs from many other languages.
